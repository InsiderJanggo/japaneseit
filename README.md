# JapaneseIT

Work in progress japanese dictionary 

![home page](https://github.com/InsiderJanggo/japaneseit/blob/main/assets/homepage.png)

## Installation

make sure to install [Nodejs](https://nodejs.org) and [MySQL](https://www.mysql.com)

for the server
```bash
cd server
npm i --save
```
for the client
```bash
cd client
npm i --save
```
